## 📝 프로젝트 소개 (Introduction)

> **"화면을 마주하는 순간, 배움의 거리는 사라집니다."**
> 
> **페이스터디(FaceStudy)**는 선생님과 학생을 이어주는 가장 완벽한 비대면 학습 공간입니다. 
> 단순한 일방향 영상 송출을 넘어, 실시간 화이트보드 협업, 화상 통화, 그리고 AI 튜터링을 결합한 상호작용 중심의 교육 솔루션을 지향합니다.

본 저장소는 **FaceStudy** 플랫폼의 안정적인 실시간 동기화, 강의실 세션 관리, AI 질의응답 등의 핵심 비즈니스 로직을 구축하는 **백엔드(Backend)** 프로젝트입니다.

---

## 💡 주요 기능 (Core Features)

| 분류 | 기능 | 설명 |
| :---: | :--- | :--- |
| **🎥** | **실시간 화상 강의** | WebRTC 기반의 실시간 화상통화 및 화면공유, 대기시간을 최소화한 채팅 채널 지원 |
| **✏️** | **공유 화이트보드** | 다수의 참여자가 동시에 판서할 수 있는 캔버스와 세밀한 수업 권한 원격 제어 |
| **🤝** | **선생님-학생 매칭** | 카테고리/태그 기반 정밀 강의 검색 및 맞춤형 개인 매칭 서비스 |
| **🤖** | **QnA 및 AI 질문** | 질문/답변 게시판에 지능형 AI 모델을 접목하여 빠르고 실용적인 자가 학습 보조 |
| **📊** | **LMS (학습 관리)** | 출석부, 과제물 피드백, 공지사항을 아우르는 올인원 학사 관리 시스템 |
